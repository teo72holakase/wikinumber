# WikiNumber

**Un número. Un artículo. Siempre el mismo.**

</div>

---

WikiNumber convierte cualquier número en una coordenada estable dentro de Wikipedia. El mismo número siempre devuelve el mismo artículo — sin importar cuándo, desde dónde o en qué idioma lo busques.

### ✨ Características

- 🌗 Tema claro / oscuro con 6 acentos de color
- 🌍 4 idiomas de interfaz (ES, ES-España, EN, PT) — el artículo se traduce automáticamente
- 🔢 Algoritmo determinístico: mismo número → mismo artículo, siempre
- 🖼️ Marco de resultado con tipografía estilo Wikipedia e imagen adaptable
- 📱 Totalmente responsive

### 🚀 Stack

HTML · CSS · JavaScript puro — sin frameworks, sin build step.

### 📂 Estructura

\```
wikinumber/
├── index.html
├── style.css
├── script.js
├── translations.js
├── favicon.ico
└── _headers
\```

### 🛠️ Desarrollo local

\```bash
git clone https://github.com/teo72holakase/wikinumber.git
cd wikinumber
# abrir index.html en el navegador, sin servidor necesario
\```

### ☁️ Despliegue

Alojado en **Cloudflare Pages**, con build output directory en la raíz (`/`).
